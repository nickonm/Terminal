Linux terminal (GitBash) commands
1)	Посмотреть где я
pwd

2)	Создать папку
mkdir newfolder

3)	Зайти в папку
cd new

4)	Создать 3 папки
mkdir 1 2 3

5)	Зайти в любую папку
cd 1

6)	Создать 5 файлов (3 txt, 2 json)
touch qq.txt q2txt qq.json q1.txt q2.json

7)	Создать 3 папки
mkdir qa1 qa2 qa3

8. Вывести список содержимого папки
ls -la
dir

8)	+ Открыть любой txt файл
cat  >> qq.txt

9)	+ написать туда что-нибудь, любой текст.
Hellow Vadim
нажать <Enter>

10)	+ сохранить и выйти.
Нажать <Ctrl>+C

12) Выйти из папки на уровень выше
cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку.
mv 123.txt 456.txt /qa1

14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
cp 123.txt 456.txt /qa1

15) Найти файл по имени
find <file_name> - это поиск в папке. Если искать с ключом -name - будет искать также во вложенных папках

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
tail -f q1.txt | grep logo

17) вывести несколько первых строк из текстового файла
head -n4 q1.txt(можно и без n)

18) вывести несколько последних строк из текстового файла
tail -5 q1.txt

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
less -N -I q1.txt

20) вывести дату и время
date
